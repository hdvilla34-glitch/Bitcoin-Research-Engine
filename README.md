# Bitcoin Research Engine (BRE)

BRE (Bitcoin Research Engine) es un sistema de investigación cuantitativa diseñado para descubrir conocimiento objetivo sobre el comportamiento de Bitcoin.

## Filosofía

Las estrategias son una consecuencia del conocimiento.

Nunca el objetivo de la investigación.

## Estado

🚧 En desarrollo (v0.1)

## Objetivo

Construir una base de conocimiento científica y reproducible sobre el comportamiento del mercado.
